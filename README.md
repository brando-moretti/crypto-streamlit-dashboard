# 📊 Real-Time Crypto Dashboard

A responsive, real-time web application that tracks and visualizes cryptocurrency market data. Built entirely in Python, this dashboard extracts live data via REST API and processes it into interactive financial charts. 

This project bridges the gap between raw external data and user-friendly visualization, demonstrating practical skills in data extraction, processing, and frontend deployment.

## Key Features
* **Live Market Metrics:** Displays real-time prices, 24h percentage changes, and market capitalization for top cryptocurrencies.
* **Interactive Data Visualization:** Utilizes Plotly to render dynamic donut charts for market dominance.
* **Historical Trend Analysis:** Processes time-series data using Pandas to visualize 7-day price trends.
* **Optimized API Calls:** Implements server-side data caching to minimize redundant requests, prevent rate-limiting, and improve application performance.

## Tech Stack
* **Language:** Python
* **Frontend Framework:** Streamlit
* **Data Processing:** Pandas
* **Visualization:** Plotly
* **Data Source:** CoinGecko REST API
